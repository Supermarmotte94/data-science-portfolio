# Portfolio d'Analyste de Données

## 📊 Vue d'ensemble

Ce repository contient un portfolio complet d'analyste de données démontrant des compétences techniques et méthodologiques à travers quatre projets d'analyse de données réels.

## 🎯 Objectif

Créé pour démontrer mes compétences en analyse de données et ma capacité à mener des projets complets de bout en bout, ce portfolio illustre différentes facettes du métier d'analyste de données junior.

## 📊 Projets Inclus

### 1. Analyse Exploratoire de Données (EDA)
- **Dataset** : Students Performance in Exams (Kaggle)
- **Objectif** : Analyser les facteurs influençant les performances scolaires
- **Techniques** : Statistiques descriptives, visualisations, analyse bivariée

### 2. Segmentation Client
- **Dataset** : Mall Customer Segmentation Data (Kaggle)
- **Objectif** : Segmenter la clientèle avec K-Means clustering
- **Techniques** : Clustering, méthode du coude, analyse des segments

### 3. Analyse de Sentiments
- **Dataset** : IMDB Movie Reviews (Kaggle)
- **Objectif** : Classifier automatiquement les sentiments des critiques
- **Techniques** : NLP, TF-IDF, classification supervisée

### 4. Test A/B Statistique
- **Dataset** : A/B Test Data (GitHub)
- **Objectif** : Évaluer l'efficacité d'une nouvelle page d'atterrissage
- **Techniques** : Tests d'hypothèses, analyse de significativité

## 🚀 Comment Utiliser ce Portfolio

```bash

```

### Exécution des Analyses
```bash

# Projet 2: Segmentation
python3 customer_segmentation.py

# Projet 3: Sentiments
python3 sentiment_analysis.py

# Projet 4: Test A/B

```

### Lancement du Tableau de Bord
```bash
cd data-portfolio-dashboard
pnpm install
pnpm run dev
```

## 🎯 Insights Clés

### Projet 1 - EDA
- Les filles excellent en lecture et écriture
- L'éducation parentale influence significativement les performances
- Les cours de préparation améliorent les scores de 5-10 points

### Projet 2 - Segmentation
- 4 segments clients identifiés
- 35% de clients cibles (revenus élevés, dépenses élevées)
- Opportunité d'engagement des clients conservateurs

### Projet 3 - Sentiments
- Modèle avec 85% de précision
- Prétraitement crucial pour la performance
- Équilibre parfait des sentiments dans l'échantillon

### Projet 4 - Test A/B
- Pas de différence significative (p-value = 0.19)
- Recommandation : ne pas déployer la nouvelle page
- Nécessité d'itérer sur le design
